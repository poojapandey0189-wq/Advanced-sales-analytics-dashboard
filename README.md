
# 📊 Advanced Sales Analytics & KPI Dashboard Suite

This project features a series of interactive Excel dashboards designed to monitor sales performance.

## 🚀 Key Features
- **Speedometer (Gauge Chart):** For real-time KPI tracking with Traffic Light logic.
- **Dynamic Visuals:** Moving bar effects and Battery-style progress indicators.
- **Evaluation System:** Custom Star Rating system for employee performance.
- **Interactive Navigation:** Uses Scroll Bars and Spin Buttons to switch between data views.
- **Advanced Formatting:** Extensive use of Conditional Formatting and Data Validation.

## 🛠️ Tools Used
- Microsoft Excel (Advanced Formulas & Dashboarding)
- Data Visualization Principles
